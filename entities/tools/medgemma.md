# MedGemma

## About
[[Google DeepMind]]'s multimodal medical AI model family. The 4B version achieves 64.4% on MedQA; the 27B version achieves 93.6% on EHR-based Q&A running on a single GPU. Includes MedSigLIP vision encoder (400M params, trained on 33M+ medical image-text pairs) for medical image interpretation across chest X-ray, dermatology, fundoscopy, and histopathology.

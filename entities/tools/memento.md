# Memento

## About
A memory-based continual learning framework for LLM agents. Uses a Case Bank to store past experiences (successes and failures) and retrieves them for planning without modifying model parameters. Architecture: Planner (GPT-4.1) + Executor (o3/o4-mini) connected via MCP. Achieved 87.9% on GAIA (#1 open-source), 95% on SimpleQA, 24.4% on HLE (near GPT-5).

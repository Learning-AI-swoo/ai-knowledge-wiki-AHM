# MultiverSeg

## About
An interactive AI tool for medical image segmentation that progressively learns from user annotations. Users click/scribble/draw on images; the model accumulates a context set and gradually requires less input, eventually reaching zero-interaction automatic segmentation. No presegmented datasets or ML expertise needed.

# Magentic-UI

## About
An open-source [[Human-in-the-Loop Systems|HITL]] platform by [[Microsoft Research]] for human-AI collaboration. Supports web browsing, code execution, and file manipulation through multi-agent architecture. Features co-planning, co-tasking, Action Guard (reversibility-based action approval), and plan memory. Uses MCP (Model Context Protocol) for external tool integration.

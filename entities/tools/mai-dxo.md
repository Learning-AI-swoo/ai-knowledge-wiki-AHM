# MAI-DxO (MAI Diagnostic Orchestrator)

## About
A structured AI diagnostic system developed by [[Microsoft Research]] that uses a virtual physician panel with specialized roles (Dr. Hypothesis, Dr. Test-Chooser, Dr. Challenger, Dr. Stewardship, Dr. Checklist) to perform sequential clinical diagnosis. Achieved 85% accuracy on NEJM cases while reducing test costs by up to 70%.
